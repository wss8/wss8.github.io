1.1、位运算符

位运算符直接对bit位进行操作，其效率最高

  • & 、| 、^、~、<<、>>

左移和右移注意点

  • 左操作数必须为整数类型 ：char 和 short 被隐式转换为 int 后进行移位操作 
  • 右操作数的范围必须为： [0, 31] 
  • 左移运算符 << 将运算数的二进制位左移 
    • 规则：高位丢弃，低位补0 
  • 右移运算符 >> 把运算数的二进制位右移 ：
    • 规则：高位补0或1，低位丢弃
      • 有符号整数，算术右移（sar），高位补符号位；如：int x = -1; x >> 1 -> ffffffffh >> 1 -> ffffffffh
      • 无符号整数，逻辑右移（shr），高位补0；如：unsigned x = -1; x >> 1 -> ffffffffh >> 1 -> 7fffffffh

注意事项：

  • 标准C未定义右操作数为负数情况，结果要根据编译器厂商，gcc编译器认为左移-1即是右移一位
  • 加法优先级大于左右移，所以推荐写法：0x1 << (2 + 3)，而不是0x1 << 2 + 3
  • 位运算符只能用于整数类型 ，左移和右移运算符的右操作数范围必须为[0, 31] 
  • 左移n位相当于乘以2的n次方，但效率比数学运算符高 
  • 右移n位相当于除以2的n次方，但效率比数学运算符高 
#include <stdio.h>  

int main()  
{  
    printf("%d\n", 3 << 2);  // 3 * 4 = 12
    printf("%d\n", 3 >> 1);  // 3 / 2 = 1
    printf("%d\n", -1 >> 1); // ffff ffff >> 1 补符号位，ffff ffff  
    printf("%d\n", 0x01 << 2 + 3);  // 0x01 << 5 -> 1 * 2^5 = 32

    printf("%d\n", 3 << -1); // oops!  

    return 0;  
}

1.2、逻辑运算符

逻辑运算符

  • ||
    • 从左向右开始计算： 当遇到为真的条件时停止计算，整个表达式为真 ，所有条件为假时表达式才为假 
  • &&
    • 从左向右开始计算： 当遇到为假的条件时停止计算，整个表达式为假 ，所有条件为真时表达式才为真
    • 逻辑表达式中，&&比 || 具有更高的优先级
    • 在&&和 || 混合运算时，整个表达式被看作 || 表达式， 编译器从左向右开始计算&&表达式，当某个&&表达式的值为真时，停止计算，整个表达式的值为真
    • ++i || ++j && ++k; -> (++i) || (++j && ++k); -> (true && ++i) || (++j && ++k);
  • !
    • 只认得0，见了0就返回1。因此当其碰见的值不是0时，其结果为0。如：!0为1, !100为0, !-100为0

位运算与逻辑运算不同：

  • 位运算没有短路规则，每个操作数都参与运算
  • 位运算的结果为整数，而不是0或1 
  • 位运算优先级高于逻辑运算优先级
  • 应避免位运算符，逻辑运算符和数学运算符同时出现在一个表达式中 
  • 当位运算符，逻辑运算符和数学运算符需要同时参与运算时， 尽量使用括号（）来表达计算次序 
  • 运算符优先级：! > 算术运算符 > 关系运算符 > && > || > 条件运算符 > 赋值运算符 > 逗号运算符
#include <stdio.h>  

int g = 0;

int f()
{
    printf("In f()...\n");
    return g++;
}

int main()
{
    int i = 0;
    int j = 0;
    int k = 0;

    if (++i | ++j & ++k) // 把位运算误用为逻辑运算
    {
        printf("Run here...\n");
    }

    printf("%d\n", i--); // 1
    printf("%d\n", j--); // 1
    printf("%d\n", k--); // 1

    ++i || ++j && ++k; // 短路

    printf("%d\n", i); // 1
    printf("%d\n", j); // 0
    printf("%d\n", k); // 0

    if (g || f() && f()) // In f()...  由于func()返回g后再g++, 所以第一个func()就为0
    {
        printf("In if statement: %d\n", g);
    }

    printf("In main(): %d\n", g); // In main(): 1

    return 0;
}

1.3、位运算实现加法

计算机只认识0和1，也只会做位运算，连最基本的加、减、乘、除都无法直接计算，而是要通过位运算间接来实现
通过位运算来实现四则运算：

1.4、位运算的各种秀

1.4.1 示例一

  • 与：任何数与0相与结果都是0，与同等位数的1相与都是他自己
  • 熟记：$$2^3 = 8;\ 2^4 = 16;\ 2^5 = 32;\ 2^6 = 64;\ 2^7 = 128;\ 2^8 = 256; \ 2^{10} = 1024; $$
  • 使用 & 判断奇偶数
    • 偶数：10 ，100 ，110 ，1000， 1010， 偶数的最低位总是0
    • 直观写法：n % 2  == 0  
    • 高效写法：n & 1  == 0
  • 使用&判断一个数是不是2的整数次方  
    • num & 31(``0001 1111) 代表取非32倍数的部分
    • 01    10    100  1000  10000 分别与上 00 01 011 0111，若&后的结果为0,可以说明为2的整数次方 
    • 得到结论：(num & (num - 1)) == 0, num > 0 ，即可说明num为2的整数次方
  • leetcode 342 - 4的幂
#include <iostream>
#include <math.h>
using namespace std;

// 求解4的幂 https://leetcode-cn.com/problems/power-of-four/

// 解法1
// 4的n次方表现为 4 * 4 * 4， 所以只要不断除以4最终结果肯定为1
bool func1(int num)
{
    if(num <= 0) return false;

    while (num % 4 == 0) num /= 4;

    return num == 1;
}
// 解法2
// 计算出所有结果，放到容器，检测num是否在容器中
// 略

// 如果数字为 4 的幂 x = 4^ax=4
bool func2(int num)
{
    if(num <= 0) return false;

    return int(log(num)/log(4)) % 2 == 0;
}

// 解法3
// 4 的幂表现为 10101010..
bool func3(int num)
{
    return (num > 0) && ((num & (num - 1)) == 0) && ((num & 0xaaaaaaaa) == 0);
}

// 解法4
bool func4(int num)
{
    return (num > 0) && ((num & (num - 1)) == 0) && (num % 3 == 1);
}

int main(void)
{
    cout << func1(16) << endl << endl;;

    cout << int(log(64)/log(4)) << endl;

    return 0;
}

1.4.2 示例二

异或的性质

  • 一个数 异或 0 结果就是他自己 ，num ^ 0 = num;  
  • 一个数 异或 自己 为0， num ^ num = 0;  
  • 一个数 连续异或两次 也是自己(由前两条性质推出)， num ^ num ^ num = num;

使用异或交换变量

  • a = a ^ b;
  • b = a ^ b; // b = a ^ b ^ b = a
  • a = a ^ b; // a = a ^ b ^ a = b

找不重复数

  • 问题：一组整形数字，只有一个数字是奇数个，其他数字都是偶数个，找出这个奇数个的数
  • 方法：所有的数一起异或，由于其他数字都是偶数个，所以异或的结果为0，而奇数个的数会多余一个，循环结束找到这个数

计算num^n

  • 取得十进制数2021的每一个数字： 不断的取余10获得个位数，再整除10排除个位数
  • 取得而进制数1011的每一个数字： 不断的&1，获得最右边的数，再右移1位
#include <iostream>

using namespace std;

void func1(unsigned int num)
{
    while(num > 0)
    {
        cout << num % 10;
        num /= 10;
    }
}

// 计算num^n(n个num累乘)
unsigned int func2(unsigned int num, int n = 2)
{
    int ret = 1;

    for(int i = 1; i <= n; i++)
    {
        ret *= num;
    }

    return ret;
}

// n个num累乘 (num^n)
unsigned int func3(unsigned int num, int n = 2)
{
    int ret = 1;

    // 假设 num为2， n为10101B，10101B个2累乘
    // 2^10101b = 2^1b * 2^100b * 2^10000b 发现遇到1就将最终结果累乘
    // 
    while(n > 0)
    {
        if((n & 1) == 1)
        {
            ret *= num; // 累乘 2^1 * 2^4 * 2^16
        }

        num *= num;

        n = n >> 1;
    }

    return ret;
}

int main3()
{
    func1(2021);

    cout << endl;
    cout << func2(2, 4) << endl;
    cout << func3(2, 4) << endl;
    return 0;
}

找出(0, num]中的最大2次幂

#include <iostream>
#include <string>
using namespace std;

// 下面是最简单直观的方法，暴力
unsigned int func_1(unsigned int num)
{
    unsigned int ret = 0;

    while (num > 0)
    {
        if ((num & (num - 1)) == 0) // 括号一个也不能少
        {
            ret = num;
            break;
        }

        //cout << num << endl;
        num--;
    }

    return ret;
}

// 10101100获取10000000：可以把所有位变为1，再加一右移一位 (11111111 + 1) >> 1 = 10000000
// 下面是推理过程
// num == 10101100
// num |= (num >> 1); // 11010110 确保左边2位为1
// num |= (num >> 2); // 11111010 确保左边4位为1
// num |= (num >> 4); // 11111111 确保左边8位为1
// 下面是结论
// 2b  1次右移
// 4b  2次右移
// 8b  3次右移
// 16b 4次右移
// 32b 5次右移
unsigned int func_2(unsigned int num)
{
    int size = sizeof(num) * 8; // 总位数
    int c = 1;
    for (int i = 0; c < size; i++) {
        num |= (num >> i);
        c *= 2;
    }
    return (num + 1) >> 1;
}
int main()
{
    cout << func_1(8888) << endl; // 8192
    cout << func_2(8888) << endl; // 8192
    return 0;
}

假如有近100个不重复的正整数，范围在[1-100]之间，但是现在缺失了一些数，要确定缺失的数据。
若改为40亿个不重复无序数中，判断一个数判断是否40亿个数中？

#include <iostream>
#include <string.h>
using namespace std;

// 本节参考：https://blog.csdn.net/recsysml/article/details/17922303，看了一下源代码似乎不兼容标准C, 所以有所改动

// static int a[100/32] = {0};
// 100个数，所以需要100b，100b=12.5B，所以需要4个int，每个int 32b, 共128位  Hex: 00000000 00000000 00000000 00000000
// a[n>>5] == a[n/32] 可以定位n映射的位置在哪个int中，如 0-31映射的位落在a[0]中，32-63映射在a[1]中
// n & 31 == n % 32 可以定位n映射所在的int中的偏移位。如 32和36映射在a[1]（000...10001），这个偏移具体应该在哪个位置，可以自己定义.例如：这里定义从右往左数，0和4为下标
// 当需要某一个位为1时，只需要异或这个位为1的数就可以使得该位为1，如下标4，对应10000；下标3，对应1000；下标n，对应1<<n；

template <int N>
class BitSet
{
private:
    int* m_arr;
    const int SIZE;
public:
    BitSet() : SIZE(N / 32)
    {
        m_arr = new int[SIZE];
        memset(m_arr, 0, SIZE);
    }
    // 将第n位设置为1
    void set(int n, bool value = 1)
    {
        if(value)
            m_arr[n >> 5] |= (1 << (n&31));
        else
            m_arr[n >> 5] &= (~(1 << (n&31)));
    }

    // 第n位是否为1，应该遵循set的规则来判断
    int test(int n)
    {
        return m_arr[n >> 5] & (1 << (n&31));
    }

    ~BitSet(){
        delete[] m_arr;
    }
};

int main()
{
    int b[100]={11,21,31,41,51,61,71,81,91,
                2,12,22,32,42,52,62,72,82,92,
                3,13,23,33,43,53,63,73,83,93,
                4,14,24,34,44,54,64,74,84,94,
                5,15,25,35,45,55,65,75,85,95,
                6,16,36,56,66,76,86,96,
                7,17,27,37,47,57,67,77,87,97,
                8,18,28,38,48,58,68,78,88,98,
                9,19,29,39,49,59,69,79,89,99,
                10,100};

    BitSet<100> bitset;

    for(int i=0; i<100; i++) // 根据数组里面的数据，将相应的比特位置为1
    {
        bitset.set(b[i], 1);
    }

    bitset.set(b[0], 0);

    for(int i=1; i<=100; i++)
    {
        if(bitset.test(i)==0)
        {
            cout << i << " is not exist" << endl;
        }
        else
        {
            cout << i << " is exist" << endl;
        }
    }


    return 0;
}
